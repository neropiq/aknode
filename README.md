# aknode

## Overview

This  library is the node server for Aidos Kuneen for step2, 
including:

* Messaging between nodes (Testing)
* Consensus / Validator (Not Yet)
* Aidos Explorer (Testing)
* RPC APIs with Wallet Service (Testing)
